# Tip-gcp (Google Cloud Platform)

## 명령어 
```bash
// 권한 계정 리스트 보기
gcloud auth list

// 설정된 프로젝트명 보기
gcloud config list project

// 프로젝트명 새로 설정
gcloud config set project <PROJECT_ID>

// Instance zone 설정
gcloud config set compute/zone us-central1-a

// Instance region 설정
gcloud config set compute/region us-central1
```
