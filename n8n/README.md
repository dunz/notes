## 텔레그램 봇 chat id 확인하는 법
텔레그램을 전송하려면 chat id가 필요한데 확인하기 까다롭다  
아래 주소를 접속한 상태에서 챗봇에 메시지를 보내면 확인 가능하다

```
https://api.telegram.org/bot(봇token)/getUpdates
```
